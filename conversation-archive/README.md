# 對話記錄檔案庫 (Conversation Archive)

## 目的
大佬，呢個 structure 係用嚟記錄我哋所有重要嘅對話內容，方便你以後快速查找同回顧。

## 結構構

```
conversation-archive/
├── 2025-02-25/
│   ├── ambient-discussions.md
│   ├── financial-planning.md
│   ├── skills-money-making-ideas.md
│   ├── girlfriend-relationship.md
│   └── daily-tasks.md
├── 2025-02-26/
│   └── (當日內容)
└── summaries/
    └── monthly-summary.md
```

## 文件說明

### 1. **ambient-discussions.md**
- 記錄所有關於 Ambient 小夜燈嘅討論
- 包括：產品、營銷策略、IG caption、客戶反饋等

### 2. **financial-planning.md**
- 收入來源（Uber、炒股、炒房）
- 支出（生活費、房租）
- 目標（電單車、卡丁車、搵女）

### 3. **skills-money-making-ideas.md**
- 賺錢 skills（股票、外賣、網絡行銷等）
- 學習資源（投資、貿易、金融）
- 每個方向嘅具體操作方法

### 4. **girlfriend-relationship.md**
- 重要日期（紀念日、生日）
- 關於佢嘅喜喜好、習慣
- 衝突記錄同解決方案

### 5. **daily-tasks.md**
- 每日待辦事項
- 優先級（緊急、重要、一般）
- 完成狀態追蹤

### 6. **monthly-summary.md**
- 每月重大事件
- 收支總結
- 學習同成長記錄

## 使用方式

大佬，當你需要搵之前嘅資訊時：

1. 訴我：「幫我搵下同佢傾過嘅關於 IG caption 嘅嘢」
2. 我會立即入 `conversation-archive/2025-02-25/ambient-discussions.md`
3. 用搜索功能快速定位相關內容

## 優點

- **易於查找** - 按主題分類，唔使亂搵
- **時序清晰** - 按日期組織，方便追蹤進度
- **可擴展** - 新增主題只需新增新文件夾
- **版本控制** - 所有更改都有 git 記錄
