# Ambient 手作工作坊 • Shopify + Instagram 自動化\n\n這個專案會把您在 Shopify 上新上架的商品自動發佈到 Instagram（Business 帳號）上，並支援排程。\n\n---\n\n## 目錄結構\n\n```.tree\nambient-shopify-ig/\n├─ README.md                # 本說明文件\n├─ requirements.txt        # Python 相依套件\n├─ setup.sh                # 一鍵安裝腳本\n├─ sync_shopify_to_ig.py   # 主程式，抓商品並貼 IG\n├─ .env.example            # 範例環境變數檔（請自行建立 .env）\n└─ utils/\n   ├─ shopify_helper.py   # 包裝 Shopify API 的輔助函式\n   └─ ig_helper.py        # 包裝 Instagram Graph API 的輔助函式\n```\n\n---\n\n## 前置需求\n\n1. **Shopify 商店**（已開通）\n2. **Instagram Business 帳號** 並已連結至 Facebook Page\n3. **Facebook Graph API** 權限（取得長期 Access Token）\n4. **Python 3.9+**\n5. **Git**（可選，用來管理程式碼）\n\n---\n\n## 步驟 1 – 取得 API 金鑰\n\n### Shopify\n- 前往 **Shopify Admin → Apps → Manage private apps**（或 **Apps → Develop apps**）\n- 建立 Private App 並勾選 **Products read/write**、**Storefront read/write** 等權限\n- 取得 **API Key**、**Password**，以及 **Store URL**（如 `https://your-store.myshopify.com`）\n\n### Instagram (Facebook Graph)\n1. 前往 **Meta for Developers**，建立新 App（Consumer 類型）\n2. 在左側選 **Products → Instagram → Set Up Instagram Basic Display**\n3. 產生 **App ID**、**App Secret**，設定 **Valid OAuth Redirect URIs**（例如 `https://localhost/`）\n4. 透過 OAuth 取得 **短期 Access Token**（手動在瀏覽器授權）\n5. 用以下 API 交換長期 token（有效期約 60 天）:\n   ```bash\n   GET https://graph.facebook.com/v15.0/oauth/access_token?\n       grant_type=fb_exchange_token\n       &client_id={APP_ID}\n       &client_secret={APP_SECRET}\n       &fb_exchange_token={SHORT_LIVED_TOKEN}\n   ```\n6. 把取得的 **長期 Access Token** 寫入 `.env`。\n\n---\n\n## 步驟 2 – 建立專案環境\n\n```bash\n# 1. 解壓或 clone 此 repo\ngit clone <repo-url> ambient-shopify-ig   # 若是 zip，直接解壓即可\ncd ambient-shopify-ig\n\n# 2. 複製範例 .env 並填入金鑰\ncp .env.example .env\n# 用您喜好的編輯器開啟 .env，填入以下變數\n#   SHOPIFY_API_KEY, SHOPIFY_PASSWORD, SHOPIFY_STORE_URL\n#   IG_USER_ID, IG_ACCESS_TOKEN\n\n# 3. 一鍵安裝依賴 (建立 virtualenv)\nchmod +x setup.sh && ./setup.sh\n\n# 4. 測試執行（會抓取最近 24 小時內的商品）\nsource .venv/bin/activate\npython sync_shopify_to_ig.py\n```\n\n---\n\n## 步驟 3 – 排程自動同步\n\n### 使用 cron（Linux/macOS）\n```cron\n# 每 4 小時執行一次（可自行調整 SYNC_INTERVAL_MINUTES）\n0 */4 * * * cd /full/path/to/ambient-shopify-ig && \
    /full/path/to/ambient-shopify-ig/.venv/bin/python sync_shopify_to_ig.py >> /var/log/ambient_sync.log 2>&1\n```\n\n### 使用 `schedule`（Python）\n如果您想在程式內部自行排程，可改 `sync_shopify_to_ig.py` 加入以下簡易範例：\n```python\nimport schedule, time\n\nschedule.every(4).hours.do(main)\nwhile True:\n    schedule.run_pending()\n    time.sleep(1)\n```\n\n---\n\n## 步驟 4 – 常見問題與除錯\n\n1. **找不到 IG_USER_ID**\n   - 在 Facebook Page 的 **Settings → Instagram** 裡，可看到 Instagram Business Account ID。\n2. **Access Token 失效**\n   - 長期 token 仍會在 60 天後失效，請設定一個提醒（例如使用 Google Calendar）定期重新換 token。\n3. **Shopify 商品沒有圖片**\n   - 程式會跳過沒有圖片的商品，您可以在 Shopify 後台先上傳主圖。\n4. **發佈的貼文顯示錯誤**\n   - 確認圖片尺寸符合 Instagram 要求（1080×1080 或 1080×1350 JPEG < 8 MB）。\n\n---\n\n## 步驟 5 – 行銷小技巧（README 內同樣列出）\n\n- **最佳貼文時間**：香港使用者活躍於 19:00‑21:00 與 08:00‑10:00。\n- **常用 Hashtag**：`#handmade #ambientHK #hongkongart #makersgonnamake #supportlocal #craftsman #design #shoplocal`（每篇 5‑10 個）\n- **Story 互動**：發布後 30 分鐘內回覆留言，提高演算法曝光。\n- **上滑連結**：若已開通 Instagram Shopping，可在貼文中直接標記商品，讓觀眾點擊直接前往 Shopify。\n\n---\n\n## 目錄說明（程式碼）\n\n### paths.append(fname)\n    return paths\n```\n\n### utils/ig_helper.py\n```python\nimport os, requests\n\nBASE_URL = 'https://graph.facebook.com/v15.0'\n\ndef _auth():\n    return {'access_token': os.getenv('IG_ACCESS_TOKEN') }\n\ndef upload_image(image_path):\n    # Instagram Graph API 只接受公開 URL，這裡示範直接上傳（實務上建議先上傳至 S3 再提供 URL）\n    url = f'{BASE_URL}/{os.getenv('IG_USER_ID')}/media'\n    files = {'image_url': open(image_path, 'rb') }\n    data = { **_auth(), 'caption': '' }\n    r = requests.post(url, data=data, files=files)\n    r.raise_for_status()\n    return r.json()['id']\n\ndef publish_media(media_id, caption):\n    url = f'{BASE_URL}/{os.getenv('IG_USER_ID')}/media_publish'\n    payload = { 'creation_id': media_id, 'caption': caption, **_auth() }\n    r = requests.post(url, data=payload)\n    r.raise_for_status()\n    return r.json()\n```\n\n### sync_shopify_to_ig.py\n```python\n#!/usr/bin/env python3\nimport os, json, logging, datetime\nfrom datetime import timezone, timedelta\nfrom utils.shopify_helper import init_shopify, get_new_products, download_product_images\nfrom utils.ig_helper import upload_image, publish_media\nfrom dotenv import load_dotenv\n\nload_dotenv()\nlogging.basicConfig(level=logging.INFO, format='%(asctime)s %(levelname)s %(message)s')\nSTATE_FILE = 'last_sync.json'\n\ndef load_last():\n    if os.path.exists(STATE_FILE):\n        with open(STATE_FILE) as f:\n            return json.load(f).get('last_iso')\n    return (datetime.datetime.now(timezone.utc)-timedelta(days=1)).isoformat()\n\ndef save_last(ts):\n    with open(STATE_FILE,'w') as f:\n        json.dump({'last_iso': ts}, f)\n\ndef main():\n    init_shopify()\n    since = load_last()\n    logging.info(f'Fetching products after #{p.id} – {p.title}')\n        img_paths = download_product_images(p, f'/tmp/ambient_{p.id}')\n        caption = f"{p.title}\n{p.body_html or ''}\n#handmade #ambientHK"\n        media_id = upload_image(img_paths[0])\n        publish_media(media_id, caption)\n        logging.info('Posted to Instagram')\n    newest = max(p.created_at for p in products)\n    save_last(newest)\n    logging.info('Sync completed')\n\nif __name__ == '__main__':\n    main()\n```\n\n---\n\n## 結語\n\n只要依照以上步驟完成設定，您的 Shopify 商品將會自動在 Instagram
